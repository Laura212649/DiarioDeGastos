DIÁRIO DE GASTOS

Aplicação Java para gerenciar seus gastos semanais e mensais diretamente pelo Terminal. Permite cadastrar, editar, remover e consultar gastos, com persistência dos dados em arquivo texto.

FUNCIONALIDADES

Adicionar gasto: registre descrição, categoria, valor e data.

Editar gasto: altere informações de um gasto já cadastrado.

Remover gasto: exclua um gasto da lista.

Consultar gastos: filtre e visualize gastos por semana ou mês.

Persistência: salve e carregue os dados automaticamente a partir de um arquivo texto.

OBSERVAÇÕES

Para apagar todos os registros, exclua o arquivo gastos.txt da pasta do projeto.

O programa é inteiramente em modo texto (console/terminal).

Caso encontre problemas, verifique se possui permissão de leitura e escrita na pasta do projeto.
